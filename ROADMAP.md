## Feature proposte

### Placeholder da completare

- [ ] **Database** — Mini DB manager per note strutturate, contatti extra, asset di progetto
- [ ] **Reports** — Report PDF/export con riassunto: ore lavorate, fatturato, task completati per periodo
- [ ] **Analytics** — Grafici avanzati: revenue per cliente, ore per progetto, trend mensili, profittabilità

### Gestione progetti

- [ ] **Note/Documenti** — Editor rich text per appunti, meeting notes, specifiche di progetto
- [ ] **Scadenze/Calendario** — Vista calendario con deadline task, scadenze fatture, milestone
- [ ] **Notifiche** — Fatture in scadenza, task scaduti, workflow GitHub falliti
- [ ] **Activity Log** — Timeline di tutte le azioni su un progetto (chi ha fatto cosa e quando)
- [ ] **Duplica Progetto** — Template da progetto esistente per nuovi clienti simili

### Fatturazione

- [ ] **Template Fatture** — Template riutilizzabili per fatture ricorrenti (es. manutenzione mensile)
- [ ] **Costi/Spese** — Tracciare spese per progetto (hosting, domini, licenze) oltre alle ore

### Clienti

- [ ] **Contatti Multipli** — Più contatti per cliente (tecnico, admin, billing)
- [ ] **Dashboard Cliente** — Vista aggregata per singolo cliente (tutti i progetti, fatture, ore)

### Dati

- [ ] **File Upload** — Upload file locali per progetto (contratti, brief, mockup), non solo GitHub
- [ ] **Export Dati** — CSV/Excel export di ore, fatture, clienti
